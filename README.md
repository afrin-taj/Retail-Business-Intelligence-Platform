# Retail Business Intelligence & Performance Analytics Platform

## Project Overview

This project focuses on analyzing retail business performance using Python and Power BI. The objective is to identify key revenue drivers, evaluate outlet performance, understand customer preferences, and generate actionable business insights through interactive dashboards.

The project includes data cleaning, exploratory data analysis (EDA), KPI development, and business intelligence dashboard creation.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook
* Power BI

---

## Dataset Features

The dataset contains retail product and outlet information, including:

* Item Type
* Item Fat Content
* Item Weight
* Item Visibility
* Sales
* Rating
* Outlet Type
* Outlet Size
* Outlet Location Type
* Outlet Establishment Year

## Key Insights

* Fruits & Vegetables generated the highest revenue.
* Snack Foods emerged as the second-largest contributor to sales.
* Low Fat products accounted for the majority of total revenue.
* Supermarket Type1 outlets generated the highest sales.
* Tier 3 locations outperformed Tier 1 and Tier 2 locations.
* Medium-sized outlets achieved the strongest overall performance.

---

## Business Recommendations

* Expand inventory for high-performing product categories.
* Increase focus on Tier 3 market expansion.
* Continue promoting Low Fat product offerings.
* Optimize outlet strategies based on outlet size performance.
* Leverage insights for data-driven business decision-making.
